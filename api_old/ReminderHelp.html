Registration

 

  • User has to do registration first by opening the application.
  • And if user has already registered previously then the user has to login by giving the registered email id and password and then click on the Log in button..
  • For the first time registration, user has to enter their first name,last name,email, mobile and then click on the sign up button.
  • After successful registration one otp will send to the given mobile number.
  • User has to verify the number by entering as password and then registration is successful.
  • In the login page there is forgot password button .If the user forgets the password then user can enter a verified email or mobile number to get the otp to reset the password.

 Navigation

  • The navigation page includes different menu buttons.
  1. Home
  2. Users
  3. My calender
  4. Add reminder
  5. My Note
  6. Reports
  7. Contact
  8. Notification
  9. Help Guide
  10. Change Password
  11. Log Out

   Home

  • In home page ,there is 4 categories of the task.i.e Pending task,Deadline crossed task,Completed task and My task.
  • The task which are created by user or admin will show in pending task list.User who has created the task can only edit or update the task.
  • The task which have crossed the assigned deadline by the user will show in the Deadline crossed task list.
  • The task in which the assigned user has given reply will show in the completed task list.In the completed task list the admin can give rating and and can close the task.
  • The task which are created by the user will show in the My task list.User can edit the update the task from it.

   

User

  

  • It will be a list of users having action,name,contactno,email,status fields.
  • Status is for verified and not verified.
  • Here user can add new participant by entering first name,last name,email and mobile.
  • Here user can create a group by entering the group name and selecting the corresponding participant fro the list.

My Calender

  • The list of task will show in the calender in the mentioned dates.
  • The different priority reminder will show in different colors

Add Reminder

             

  • In the add reminder section user will set a reminder by adding reminder type,title,description,from and to date,,group,users etc.
  • In the group user has to select a group
  • From the group user has to select a group.According to the group the respective participant list will show in the Users section.
  • In the user field user has to enter or select user name .
  • User can set the reminder interval from the interval drop down.
  • Here user can set the reminder priority and set the reminder.

My Note

  • In this section two text field would be there one is title and other is description.User has to fill both the field and click submit button to add a note.
  • The added note will show in a list below.User can delete the note by clicking on the delete button from the list.

Reports

  • The reports list will be there along with the required filter option in the top.
  • User can filter the report according to their requirements.
  • From the report user can see the report by searching the title and participant name.
  • User can view the report details from the list by clicking on the view icon and can update the data.

Contact

  • The list having contactno,name,email call button will show.user can call  directly by clicking on the given contactno.
  • From the below plus button user can create the group here also.

Notification

  • In this section user can add notification by entering title,date and message.Then user can search the notification by entering the date.
  • This functionality only works locally.

Help info

  • In the help info section we will redirect to the user guide.

Change password

  • In the change password page user can change the password by entering the current password and new password.
  • Then user has to confirm the new password and submit .Then the password will change successfully.

Log Out

By clicking on the log out button user will successfully log out from the application.
